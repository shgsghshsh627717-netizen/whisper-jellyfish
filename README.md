# 🪼 水母语音输入 · whisper-jellyfish

> 在 Mac 上，按一下快捷键就能用「说话」代替「打字」——在任何输入框里，说完话文字自动出现。
> **完全本地运行、永久免费、不需要任何 API key、不联网也能用。** 还附带一只会跟着你说话飘动的桌面水母 🪼

这是给**完全不懂编程的人**也能装得上、用得起来的版本。如果你能双击文件、会在系统设置里点开关，就能用。

---

## ✨ 它能做什么

- **到处都能用**：Claude、微信、浏览器、备忘录……任何能打字的地方，按 `Ctrl+F` 说话，文字自动落到光标处
- **带标点**：自动断句加标点，方便你检查
- **本地免费**：用 Apple 芯片本地跑 Whisper 模型（`mlx-whisper`），不花一分钱、不上传你的声音
- **可爱的水母指示器**：
  - 菜单栏一只小水母 + 桌面一只可拖动的大水母
  - 空闲时打瞌睡、录音时举花球欢呼、转录时捧爱心——一眼就知道现在是什么状态
  - 点一下桌面水母 = 按 `Ctrl+F`

---

## 💻 适用环境

- **Mac（Apple Silicon：M1/M2/M3/M4）** —— 本地转录用的是苹果芯片加速
- macOS 系统自带的终端就行
- 需要 **Python 3.12 或更新版本**（没有的话下面会教你装）

---

## 🚀 安装（三步，照着做就行）

### 第 0 步：装 Python（如果已经有就跳过）

打开「终端」（在 启动台 → 其他 → 终端），输入 `python3 --version` 回车。
- 显示 `Python 3.12` 或更高 → 已经有了，跳到第 1 步
- 提示找不到，或版本低于 3.12 → 去 [python.org 下载页](https://www.python.org/downloads/) 下载安装最新版，装好再继续

### 第 1 步：下载本项目

- 在本页面点绿色的 **「Code」按钮 → Download ZIP**，下载后**解压**
- 把解压出来的文件夹放到一个固定位置（比如「文稿」里），**之后别再移动它**

### 第 2 步：双击 `install.command`

- 打开刚解压的文件夹，**双击 `install.command`**
- 会弹出一个终端窗口自动安装（首次要几分钟，装好会提示「安装完成」）
- 如果提示「无法打开，因为来自身份不明的开发者」：**右键点 `install.command` → 打开 → 再点打开**

> 安装做了什么：建好运行环境、装好依赖、在你的**桌面生成一个「水母语音输入」图标**。

### 第 3 步：给两个系统权限（只需一次）

macOS 出于安全，需要你手动授权两样东西：

**① 辅助功能**（让它能监听 `Ctrl+F` 快捷键、把文字打进输入框）
- 系统设置 → 隐私与安全性 → **辅助功能**
- 把 **Terminal（终端）** 和 **Python** 都加进去并打开开关
- 加 Python 的方法：点 `+` → 按 `Cmd+Shift+G` → 粘贴安装完成时屏幕上显示的那行路径 → 打开

**② 麦克风**（让它能录你的声音）
- 不用手动找，**第一次按 `Ctrl+F` 录音时系统会自动弹窗**，点「允许」即可

## 🎙️ 怎么用

1. **双击桌面「水母语音输入」** 启动 → 会弹出一个终端窗口，**别关它**（关了就停了，最小化放一边即可）
2. 切到任何想打字的地方，点一下输入框
3. 按 **`Ctrl+F`** 开始录音 → 说话 → 再按一次 **`Ctrl+F`** 停止
4. 等一两秒，文字自动出现在光标位置 ✨

**小水母怎么玩：**
- 桌面上的水母可以**拖**到任意位置（它会记住）
- **点一下**水母 = 按 `Ctrl+F`（开始/停止录音）
- 看它的表情就知道状态：打瞌睡=空闲，举花球=正在听你说，捧爱心=正在转文字

**想关掉：** 回到那个终端窗口按 `Ctrl+C`，或直接关窗口。

## ❓ 常见问题

**按 Ctrl+F 没反应？**
→ 多半是「辅助功能」权限没给全。确认 Terminal 和 Python 都加进去并打开了开关；改完权限要**关掉终端窗口重新双击启动**。

**提示「没有可用的音频输入设备」？**
→ 检查麦克风是否插好；首次录音的麦克风授权弹窗有没有点「允许」。

**转录出来没有标点 / 有奇怪的乱码？**
→ 本项目已内置标点优化。如果异常，确认你用的是本仓库的最新版本。

**它会把我的声音上传到网上吗？**
→ **不会。** 纯本地运行，声音只在你电脑里处理。录音存档也只存在本地（已默认不上传、不外泄）。
